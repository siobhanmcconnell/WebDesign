Welcome to Crochet!: Getting Started

It's never too late to get a new hobby, and crochet is a great way to have fun, relax, and gain some cool new trinkets. In this post, we'll explore everything you need to get started crocheting, from tools to first projects. If you're a beginner, this is a great place to start your crochet journey!

This Month's Crochet Challenges

Learn a new stitch

Single and double crochet are great, but there are so many stitches to learn and so little time. This month, look up a new stitch and figure it out!

Weave in your ends

We all have that project that's 99% done, but it just sits on the shelf because you haven't woven in the ends. No more excuses! This is your sign to get it over with.

Find a new pattern

Lets be real. The venn diagram between hoarders and crocheters is a circle. Feed the hunter gatherer within you and treat yourself to a new pattern.

Post an in-progress piece

Every project goes through an ugly phase, but it's an important part of the process. Instead of posting the perfect finished product, show the world something you're working on!
